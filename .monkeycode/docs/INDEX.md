# 校园二手书交易平台 文档

## 项目概述

校园二手书交易平台是一个专为校内师生打造的二手书交易Web应用，旨在提供安全、便捷的书籍流通环境。平台支持二手书的发布、检索、在线交易与即时沟通，帮助用户快速处理闲置书籍并获取所需资料。

**目标用户**: 校内师生（学生/老师）

**核心价值**:
- 安全可信的交易环境，保护买卖双方权益
- 高效便捷的书籍流通，减少资源浪费
- 智能推荐系统，帮助用户发现所需书籍
- 完整的交易流程管理，清晰的订单状态流转

## 技术选型

| 类别 | 选择 | 理由 |
|------|------|------|
| 前端框架 | Vue3 + JavaScript | 渐进式框架，响应式设计，支持PC和移动端 |
| UI组件库 | ElementPlus | Vue3官方推荐组件库，简洁美观 |
| 后端框架 | SpringBoot + MyBatis | 成熟稳定，快速开发RESTful API |
| 数据库 | MySQL 8.0 | 关系型数据存储，可靠性高 |
| 缓存 | Redis | 会话管理、Feed流推送、热门排序 |
| 部署 | Docker + Nginx | 容器化部署，支持高并发 |
| 推荐算法 | Redis Feed流 + 协同过滤 | 实现个性化推荐和内容推送 |

## 核心功能

### 用户模块
- **目的**: 用户注册登录和个人信息管理
- **描述**: 支持手机号/邮箱双渠道注册登录，密码找回与修改，个人资料管理（头像、昵称、联系方式等）

### 商品模块
- **目的**: 二手书信息发布与管理
- **描述**: 支持多图上传、标题、简介、价格、新旧程度、分类等属性，包含商品审核、状态管理（在售/预订中/已售）

### 交易模块
- **目的**: 完成书籍购买流程
- **描述**: 购物车管理、订单创建、模拟支付、订单状态流转（待付款/待发货/待收货/已完成/已取消）、超时自动取消

### 评价模块
- **目的**: 买卖双方互评机制
- **描述**: 支持评分与文字评价，评价审核与展示

### 消息模块
- **目的**: 用户间沟通
- **描述**: 站内私信/留言板、文字图片交流、消息中心统一管理（系统通知/交易通知/私信）

### 搜索推荐模块
- **目的**: 帮助用户快速找到所需书籍
- **描述**: 关键词全文检索、条件筛选（价格区间/分类/新旧程度/卖家信用）、搜索历史、热门搜索、Feed流推荐、兴趣标签推荐

### 举报投诉模块
- **目的**: 维护平台秩序
- **描述**: 违规商品举报、用户投诉、建议反馈

### 管理后台
- **目的**: 平台运营管理
- **描述**: 用户管理、商品审核与管理、订单管理、评价管理、举报投诉处理、系统公告管理、分类管理、操作日志、系统监控
